Scfee

Extraire des données de géolocalisation à partir de fichiers images et les localiser.

Bienvenue à la page de SCFEE.

Nous vous présentons un script Médico-légal pour extraire les données brutes GPS, calculer les positions de longitude et lattitude afin de les intégrer dans un programme de cartographie en ligne pour tracer les emplacements des photos. Le script utilise le PIL pour extraire systématiquement des données EXIF des photos, y compris les données GPS lorsqu'elles sont disponibles et traiter des types de données plus complexes.

Comment l'utiliser le script d'injection de code

Dans un prompt de ligne de commande, lancez le script comme suit:

C:>Python SCp_Extracteur_GPS.py-d c:\images\ -c c:\images\Résultats\ -l c:\images\Log\

Collez les coordonnées, sélectionnées du résultat de l'exécution du programme, dans le site Web en ligne, http://www.darrinward.com/lat-long/ et soumettez-y les valeurs.

Avant de commencer

installez la bibliothèque PIL dans votre machine Linux à partir du lien suivant: http://effbot.org/downloads/Imaging-1.1.7.tar.gz

installez la bibliothèque PIL dans votre machine Windows à partir du lien suivant: http://effbot.org/downloads/PIL-1.1.7.win32-py2.7.exe
